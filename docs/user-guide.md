# User Guide

Welcome to the **Universal Document Engine (UDE)** User Guide!

This guide will walk you through the key concepts and usage patterns of UDE to generate stunning documentation portals for your software.

## 🚀 Getting Started

Universal Document Engine parses Doxygen XML outputs and transforms them into modern, responsive static sites.

### Basic Usage

To run the generator, use the command-line interface:

```bash
python -m ude.cli --input path/to/doxygen/xml --output path/to/output/markdown
```

### Configuration

You can customize the rendering behavior by providing a configuration file in YAML or JSON format.
